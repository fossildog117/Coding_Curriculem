Learn Ruby

Ruby is a programming language that balances power and ease of use nicely to create an easy to use language that can be used for pretty much anything. By learning how to use it you are futureproofing yourself and gaining skills in problem solving and logic that will serve you well for the entirety of your existence in this dimension. We have created this tutorial to try to get you to grips with programming in an interesting way through scenarios and real world programs. We hope you enjoy it!
