Fun Strings!

A string is a normal word that can be processed by a computer. A great aspect of Ruby is that processing strings in this language is super simple!



Here we have declared a few variables as Strings and then joined them together

my_eyes = "brown"

my_hair = "black"

my_features = "I have " + my_eyes + " eyes and " + my_hair + " hair"


puts "#{my_features}"

This will make the computer print a line saying "I have brown eyes and black hair"

Ruby also has a cool feature that allows us to choose certain parts of a string if we need!


greetings = "hello world"


greetings[0] # => "h"


greetings[0,2] # => "hel"


greetings[0..2] # => "hel"


my_features[7..16] # => brown eyes
