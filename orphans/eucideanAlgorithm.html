Finding the greatest common divisor (GCD)

Introduction to Euclid's Algorithm

The Eucideal algorithm is a method used to computer the greatest common factor between two or more numbers. It is often used to simplify fractions to their simplest form and is applied to many other fields in computing and mathematics.

Algorithm

Find the GCD of A and B where A > B

  1. If A = 0 or B = 0 return 0. Otherwise go to step 2.
  2. If A > B calculate A/B, otherwise calculate B/A. Let R be the remainder. If R = 0 go to step 4
  3. If A > B, let A = R otherwise let B = R
  4. If A > B then B = GCD otherwise A = GCD

Challenge: Can you find the greatest common divisor for more than 2 numbres?

Step by step example

Find the GCD of 54 and 12

54 > 12 so compute 54/12 = 4 with remainder 6. R = 6

12 > 6 so compute 12/6 = 2 with remainder 0. R = 0

6 < 12 and R = 0 so GCD = 6
