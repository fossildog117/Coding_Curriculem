Sequential Search

Sequential search is where we search for an item by going through a list starting from the first item and scanning through the list in order until we find the item we are looking for

Sequential Search Algorithm

  1. Let i = 1 and n = the number of elements in the list
  2. If the ith element in the list is not the item we are looking for add 1 to i
  3. If i > n then stop. Otherwise repeat step 2 until we have found the element we are looking for.

A more detailed step by step example has been given below

Find the number 64 in the given list of integers


13 , 54 , 64 , 23 , 12


First Comparison

i = 1, n = 5

i = 1 is the first number in the list which is 13

Since 13 ≠ 64, i = 2

Second Comparison

i = 2, n = 5

i = 2 is the second number in the list which is 54

Since 54 ≠ 64, i = 3

Third Comparison

i = 3, n = 5

i = 3 is the third number in the list which is 64

We have found 64 at position 3 in the list
