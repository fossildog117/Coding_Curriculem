Introduction

What is Ruby?

Ruby is a popular programming language with many applications, one of its most common applications is for creating powerful websites using a framework(A base upon which developers build) called Ruby on Rails, however it can be used for pretty much anything. We chose Ruby for this guide as we believe it is a good introduction to programming making it easy for you to explore different programming paradigms within a single language, and having a straightforward and easy syntax.

About this guide

This guide is divided into chapters which you can see on the homepage. Each chapter consists of a number of lessons covering specific topics. Programming is a skill best learnt by doing it yourself so our lessons contain challenges for you to complete yourself, we have tried to use real world examples as far as possible so you can see practical uses of programming.
